# Task 1

В рамках задания был написан скрипт на Python, который выполняет пингование 10 доменов и сохраняет результаты в CSV-файл.

Для каждого домена собираются следующие параметры:

- `RTT` — время отклика
- `IP` — IP-адрес узла
- `ICMP_SEQ` — номер ICMP-пакета
- `TTL` — значение Time To Live

Результаты сохраняются в файл `result.csv`.
