# Task 10

В рамках задания был написан скрипт на Python, который выполняет DNS lookup и трассировку маршрута (traceroute) для списка доменов и сохраняет результаты в CSV-файл.

Для каждого домена собираются следующие параметры:

- IP — IP-адрес, полученный через DNS lookup
- traceroute — маршрут пакетов до узла (каждый hop разделён символом |)

Результаты сохраняются в файл `result.csv`.
