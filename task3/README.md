# Task 3

В рамках задания реализован скрипт-парсер на **Playwright**, который выполняет сбор данных с сайта с поддержкой **пагинации**.

Скрипт:
- открывает несколько страниц сайта
- собирает данные с каждой страницы
- сохраняет результат в `csv` файл
- извлекает не менее 4 параметров для каждого объекта

В качестве примера парсинга используется лента статей **Habr**.

## Что делает скрипт

Скрипт переходит по страницам ленты Habr и собирает данные о статьях.

Для каждой статьи сохраняются следующие параметры:

- `title` — заголовок статьи
- `author` — автор статьи
- `datetime` — дата и время публикации
- `views` — количество просмотров

Все собранные данные записываются в `csv` файл.

## Пагинация

В парсере реализована пагинация: скрипт проходит по нескольким страницам сайта.

Пример логики пагинации:

- 1 страница
- 2 страница
- 3 страница

Для каждой страницы формируется отдельный URL вида:

```text
https://habr.com/ru/feed/page1/
https://habr.com/ru/feed/page2/
https://habr.com/ru/feed/page3/
