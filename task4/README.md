# Task 4 

В рамках задания реализовано API-приложение на FastAPI, которое позволяет:

- запустить парсер через HTTP-запрос, передав URL в параметре `url`
- сохранить результаты парсинга в базу данных PostgreSQL
- получить данные из таблицы базы данных в формате JSON отдельным методом

База данных используется **PostgreSQL**, как требуется в задании.
