# Task 6

В рамках задания было настроено проксирование запросов через **Nginx**.

Запросы перенаправляются в Docker-контейнер с FastAPI-приложением.

Схема работы:

`localhost:8082 -> nginx:80 -> app:8081`

- контейнер `nginx_proxy` принимает запросы
- внутри контейнера Nginx слушает `80` порт
- далее запросы проксируются в контейнер `app`, работающий на порту `8081`
